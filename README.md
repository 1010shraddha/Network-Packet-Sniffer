# Network Packet Sniffer Dashboard

## PROJECT NAME
Network Packet Sniffer and Security Monitoring Dashboard

## NO. OF WEEKS
1 Weeks - To complete this project

## PROJECT SCOPE
This project focuses on real-time network traffic monitoring and analysis using Python, Scapy, and Flask. The system captures network packets, identifies protocols (TCP, UDP, ICMP), monitors DNS requests, analyzes source and destination IP addresses, and displays traffic statistics through a live web dashboard. The project aims to provide basic network visibility, traffic analysis, and security monitoring capabilities.

## OBJECTIVES
- Capture live network packets
- Analyze network protocols
- Monitor DNS queries
- Track source and destination IPs
- Generate traffic statistics
- Visualize data through a dashboard

## TECHNOLOGIES USED
- Python
- Scapy
- Flask
- HTML/CSS
- Chart.js
- Pandas

## FEATURES
